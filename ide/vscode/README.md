# CSX Language Support

Syntax highlighting and language support for CSX (C# + JSX).

## Features

- **Component Highlighting**: Highlights `component Name` syntax.
- **JSX Support**: Full highlighting for JSX tags, attributes, and nested content.
- **C# Integration**: Seamlessly switches between C# and JSX contexts.

## Installation

1. Install the `.vsix` file manually.
2. Ensure you have the [C# Extension](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp) installed.
